# INSTALL - Shell Script
## some shell script to install some software to fedora

- discord (https://discord.com/)
```sh
bash discord.sh
```
- vlang (https://vlang.io/)
```sh
bash v.sh
```
- mp3-music-listener (https://github.com/Saverio976/mp3-music-listener)
```shell
bash listiny.sh
```
